# Sin Etiquetas · Tu Podcast

Sitio web oficial de Sin Etiquetas — podcast de salud emocional y plataforma del ecosistema Fitness Emocional.

## Estructura del proyecto

```
sinetiquetas/
├── index.html          # Página de inicio
├── public/             # Assets estáticos
│   ├── logo.png        # Logo oficial SE
│   └── estudio.jpg     # Foto del estudio
├── package.json        # Configuración del proyecto
├── railway.json        # Configuración de Railway
└── README.md           # Este archivo
```

## Cómo actualizar el sitio

### Agregar/cambiar texto
Abre `index.html` en cualquier editor de texto y busca el texto que quieres cambiar.

### Cambiar imágenes
Reemplaza los archivos en la carpeta `public/` con el mismo nombre.

### Desplegar cambios
1. Guarda los cambios
2. En GitHub Desktop: escribe un mensaje y haz clic en "Commit"
3. Haz clic en "Push" — Railway desplegará automáticamente
